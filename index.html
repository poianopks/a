GPTs 무료 주식 API 연동
왕초보 따라하기 가이드

👶 완전 처음하는 분도 따라할 수 있게, 단계별로 쉽게 설명했어요!

1단계: 무료 주식 API란?

  • Alpha Vantage – 실시간 주가, 환율, 재무, 뉴스까지 폭넓게 제공.
    무료로 일 25회 요청 가능. [사이트 바로가기]
  • Finnhub – 글로벌 주가, 경제뉴스, 재무 등.
    무료로 분당 60회. [사이트 바로가기]
  • Financial Modeling Prep(FMP) – 재무제표/기업정보. [바로가기]
이 가이드에선 Alpha Vantage로 예시를 들 거예요.

2단계: 무료 API 키 발급받기

  1. Alpha Vantage 홈페이지 접속
  2. 메인화면에서 Get Your Free API Key 클릭
  3. 이메일 주소 입력 후 제출
  4. 발급된 API Key를 복사해 저장 (예: 1234ABCD5678EFGH)
    이 키는 꼭 안전하게 보관!
API 키는 내 비밀번호! 남에게 알려주면 안돼요.

3단계: 나만의 GPT 만들기

  1. ChatGPT(https://chat.openai.com) 접속 & 로그인
  2. 좌측 GPTs 클릭 → + Create GPT 클릭
  3. 이름: 주식GPT 등 자유롭게 입력
  4. 설명: “실시간 주가·뉴스·재무 조회” 등 간단히

4단계: 액션(Action) 추가로 API 연결

  1. GPT 만들기 화면에서 Configure 탭 아래로 스크롤
  2. “Actions” 또는 “Custom Actions”에서 Add Action 클릭
  3. 입력 예시
    Action Name: GetStockPrice
    Base URL: https://www.alphavantage.co/query
    Method: GET
    Query Params:
      function = GLOBAL_QUOTE
      symbol = {{사용자입력}}
      apikey = [발급받은 API 키]
  4. Authentication(인증): API Key(쿼리 파라미터로, 이름은 apikey)
  5. 모두 입력 후 Save
이렇게 연결하면 GPT가 “삼성전자 주가 알려줘” 등 질문 받을 때 직접 API에 물어봅니다!

5단계: 시스템 명령으로 GPT 행동 지정

  1. Configure 탭의 Instructions에 아래처럼 입력
    사용자가 특정 주식의 현재가를 물어보면, GetStockPrice 액션으로 실시간 시세를 가져와 대답하세요.
  2. 저장(Save)

6단계: 미리보기(Preview)로 직접 테스트!

  1. Preview 탭에서 “Apple(AAPL) 주가 알려줘” 입력
  2. GPT가 API를 호출해 아래처럼 답변:
    현재 애플(AAPL) 주가는 150.00달러입니다. (예시)
  3. 다른 종목도 테스트해 보기

7단계: 웹 브라우징으로 최신정보 얻기 (API 없이!)

  1. Configure에서 Web Browsing ON으로 변경(버튼 클릭)
  2. Preview 탭에서 “오늘 삼성전자 뉴스 알려줘” 등 입력
  3. GPT가 인터넷에서 직접 찾아서 최신정보 답변!
단, 정확한 숫자·포맷은 API 활용이 더 좋을 수 있어요.

필수 용어 쉽게 정리

  • API: 인터넷 데이터 전화선
  • API 키: 나만의 비밀번호
  • 액션: GPT가 쓰는 도구(기능)
  • 시스템 명령: GPT에게 미리 주는 행동지침
  • 엔드포인트: API가 정해준 데이터 주소
  • 쿼리 파라미터: ?뒤에 붙는 데이터 전달 값

초간단 요약! (이렇게 따라해요)

  1. 무료 API(Alpha Vantage 등) 가입 → 키 발급
  2. GPT Builder에서 GPT 만들기 → 액션 추가 → 키/주소 입력
  3. Instructions/시스템명령에 행동 지침 입력
  4. Preview에서 질문하며 테스트
  5. 간단하게 최신 뉴스는 웹브라우징 기능으로도 OK!
