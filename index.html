👶 GPTs + 주식 API 왕초보 따라하기 가이드

이 페이지를 보면, 프로그래밍 경험이 전혀 없어도 ChatGPT를 활용해 실시간 주식 시세와 최신 뉴스를 가져오는 방법을 배울 수 있어요. HTML 파일로 저장하면 언제든 열어보며 학습할 수 있습니다.

💡 용어 쉽게 설명하기
  • GPT Builder: ChatGPT에서 직접 나만의 GPT를 만드는 공장 같은 곳이에요.
  • 액션(Action): GPT에게 특정 기능(예: 주식 데이터 가져오기)을 수행하도록 추가하는 도구 상자예요.
  • API: 앱이나 서비스가 서로 데이터를 주고받는 인터넷의 전화선 같은 거예요.
  • API 키(API Key): API를 사용할 때 필요한 비밀 번호예요. 함부로 공유하면 안 돼요!
  • 시스템 명령(System Instruction): GPT에게 주는 숨은 지시 사항이에요. 사용자는 안 보이지만, GPT는 이 내용을 보고 행동해요.
  • 웹 브라우징(Web Browsing): GPT가 스스로 인터넷을 검색하는 기능이에요.

1단계: 무료 주식 API 소개 및 키 발급받기

주식 시세, 뉴스, 재무 정보 등을 가져오기 위해 먼저 무료 API에 가입하고 API 키를 발급받을 거예요.

1-1. 대표적인 무료 주식 API

  1. Alpha Vantage
    - 실시간 주식 시세(분별, 일별), 환율, 암호화폐 시세, 재무 지표, 뉴스 등 다양한 데이터를 무료로 제공해요.
    - 무료로 하루 25회 요청할 수 있어요. (더 쓰려면 유료 업그레이드 필요)
    - Alpha Vantage 홈페이지
  2. Finnhub
    - 전 세계 주식 시세, 경제 뉴스, 기업 재무정보 등을 무료로 제공해요.
    - 무료로 분당 60회 요청할 수 있어요. (초과 시 유료 플랜 필요)
    - Finnhub 홈페이지
  3. Financial Modeling Prep (FMP)
    - 기업의 재무제표(손익계산서, 대차대조표 등), 재무비율, 주식 기본정보 등을 무료로 제공해요.
    - FMP 홈페이지

1-2. Alpha Vantage API 키 발급받기

여기서는 예시로 Alpha Vantage의 API 키를 발급받는 과정을 따라할게요.

  1. Alpha Vantage 홈페이지 접속
    웹브라우저를 열고 https://www.alphavantage.co에 접속해요.
  2. "Get Your Free API Key" 클릭
    메인 화면에서 "Get your free API key" 버튼을 찾아 클릭해요.
  3. 간단한 설문 작성
    팝업 폼이 나오면, 아래 정보를 입력해요:
    - Type of project: (예: 개인 투자용) - Industry: (예: 개인) - Email: 자신의 이메일 주소 입력 - Agree to Terms 체크
  4. API 키 확인 및 복사
    제출하면 화면에 API Key: XXXXXXXXXXXXX가 나와요. 이 키를 꼭 복사해서 안전한 곳에 저장하세요!
    예시:
    API Key: 9KHXYZ123ABCD456
    이메일로도 키가 전송되니 나중에 확인할 수 있어요.

🔒 API 키는 비밀이에요! 다른 사람에게 공유하면 안 돼요.

2단계: GPT Builder에서 새 GPT 만들기

이제 ChatGPT 안에 나만의 GPT를 만들어 볼 거예요. GPT Builder를 처음 쓰는 경우라도 걱정 마세요! 아주 쉽게 설명해 드릴게요.

  1. ChatGPT 접속 및 로그인
    • 웹브라우저에서 https://chat.openai.com에 들어가요.
    • OpenAI 계정으로 로그인합니다. (계정이 없으면 회원가입부터 해주세요.)
  2. GPTs 메뉴로 이동
    좌측 사이드바에서 GPTs 또는 Explore GPTs 아이콘을 클릭해요.
  3. "+ Create GPT" 버튼 클릭
    오른쪽 상단에 있는 + Create GPT 버튼을 눌러요.
  4. 기본 정보 입력
    - Name (이름): "주식 분석 GPT" 같이 알아보기 쉬운 이름
    - Description (설명): "실시간 주가와 뉴스를 알려주는 GPT"
    - Avatar (아이콘): 원하는 이미지를 고르거나 기본 아이콘을 사용해도 돼요.
  5. "Create GPT" 클릭
    입력이 완료되면 맨 아래에 있는 Create GPT 버튼을 누릅니다. 이제 GPT가 만들어졌어요!

성공하면 새로운 GPT 편집 화면이 보일 거예요. 이 화면에서 Configure (설정)와 Preview (미리보기) 탭을 볼 수 있어요.

3단계: 액션(Action) 추가하기

GPT에게 주식 API를 호출할 수 있는 액션을 알려줄 거예요. 액션을 추가하면 GPT가 직접 API에 접속해 실시간 데이터를 가져올 수 있습니다.

3-1. Configure 탭으로 이동

새로 만든 GPT 화면에서 Configure 탭이 선택되어 있는지 확인해요.

3-2. Actions 섹션 찾기

  1. 스크롤을 아래로 내려 Actions 또는 Custom Actions 부분을 찾습니다.
  2. "+ Add Action" 혹은 "Create new action" 버튼을 클릭해요.

3-3. 액션 정보 입력하기

이제 API 정보를 입력해야 합니다. 아래 표를 보면서 따라 해 보세요!

📋 액션 설정 예시 (Alpha Vantage)
  • Action Name: GetStockPrice (원하는 이름으로)
  • Method (HTTP 방식): GET
  • Base URL: https://www.alphavantage.co/query
  • Path: (비워두거나 /query 입력)
  • Query Parameters (쿼리 매개변수):
    • function: GLOBAL_QUOTE (항상 이 값 사용)
    • symbol: {{ticker}} (사용자가 입력한 종목 기호로 대체)
    • apikey: YOUR_API_KEY (발급받은 키를 넣으세요)
  • Authentication (인증): API Key 유형 선택 후, Query Parameter 위치, apikey 이름, 그리고 YOUR_API_KEY 입력

위 설정이 완료되면 Save 버튼을 눌러 액션을 저장하세요!

3-4. 응답 응답(Response) 예시

액션이 호출되면 API가 JSON 형태로 결과를 주는데, GPT가 읽기 쉽게 해석하려면 응답 중 필요한 정보(예: 최신 주가)를 지정해주는 게 좋아요. 예시는 아래와 같습니다:

{
  "Global Quote": {
    "01. symbol": "AAPL",
    "05. price": "150.00",
    "10. change percent": "+3.70%"
  }
}

GPT Builder에서 price 필드(예: "05. price")를 price라는 이름으로 응답 스키마에 지정하면, GPT가 이 값을 캐치하여 답변에 쓸 수 있어요.

4단계: 시스템 명령(System Instruction) 추가하기

GPT에게 언제 액션을 써야 할지 알려주는 숨은 지침을 작성할게요.

  1. Configure 탭에서 Instructions 또는 System Message 부분을 찾습니다.
  2. 아래 예시처럼 입력해 보세요:
    사용자가 특정 주식의 현재 가격을 물으면, GetStockPrice 액션을 사용해 실시간 주가 정보를 가져와 답변하세요.
  3. Save 버튼을 눌러 저장합니다.

이렇게 하면 GPT가 "애플 주가 알려줘"라고 물었을 때 자동으로 액션을 호출해서 답변해 줄 거예요!

5단계: 미리보기(Preview)에서 테스트하기

이제 GPT가 잘 작동하는지 확인합니다. Preview 탭으로 넘어가세요.

  1. Preview 탭 클릭: GPT Builder 상단 또는 우측에 있어요.
  2. 질문 입력: 예를 들어 "Apple (AAPL)의 현재 주가는 얼마예요?"라고 입력해 보세요.
  3. 액션 호출 및 응답 확인: GPT가 "GetStockPrice" 액션을 호출해 최신 데이터를 가져오고, 아래처럼 답변해 줄 거예요:
    "현재 애플(AAPL)의 실시간 주가는 150.00달러입니다. (변동률: +3.70%)"
  4. 다른 종목도 테스트: 예를 들어 "TSLA 주가 알려줘"도 물어보세요.

정상적으로 정보가 나오면 모든 설정이 완료된 거예요! 🎉

6단계: 액션 없이 최신 정보 얻기 (웹 브라우징 설정)

API 설정이 복잡하다면, 웹 브라우징 기능을 켜서 GPT가 인터넷에 직접 접속해 정보를 가져오게 할 수도 있어요.

  1. Configure 탭에서 Capabilities > Web Browsing 옵션을 켬(On)으로 변경합니다.
  2. Save를 눌러 저장하세요.
  3. Preview 탭에서 예시 질문을 입력합니다. 예: "오늘 삼성전자 주가가 어떻게 돼요?"
  4. GPT가 자동으로 인터넷을 검색해 최신 정보를 찾아 알려줍니다.

이 기능은 API 키 없이도 최신 정보를 얻을 수 있지만, 정확한 수치나 형식적인 데이터는 API를 사용했을 때보다 약간 부정확할 수 있어요.

7단계: 자주 사용하는 용어 정리

지금까지 나온 용어들을 정리해 볼게요. 한번만 읽으면 나중에 헷갈리지 않을 거예요!

  • GPT Builder: ChatGPT 안에서 Custom GPT를 만드는 곳.
  • 액션(Action): GPT에게 특정 기능을 수행하도록 추가하는 도구.
  • API: 앱끼리 데이터를 주고받는 인터넷 전화선.
  • API 키(API Key): 전화선을 사용할 때 필요한 비밀번호 같은 것.
  • 시스템 명령(System Instruction): GPT 내부에서만 보이는 숨은 지시사항.
  • 웹 브라우징(Web Browsing): GPT가 스스로 인터넷 검색하기.

마무리: 초간단 따라하기 정리

  1. 원하는 무료 주식 API에 가입하고, API 키를 꼭 복사해요.
  2. ChatGPT에서 GPT Builder로 새 GPT를 만들어요.
  3. Configure 탭에서 액션을 추가하고, API 정보를 입력해요.
  4. 시스템 명령에 "주가 알려달라면 액션을 써라"라는 지침을 넣어요.
  5. Preview 탭에서 "오늘 애플 주가 알려줘" 같은 질문으로 테스트해요.
  6. 액션 없이 그냥 최신 정보를 보고 싶다면, 웹 브라우징 옵션을 켜면 돼요.

이 과정을 모두 따라왔으면, 여러분만의 주식 도우미 GPT가 완성된 거예요! 앞으로 이 GPT에게 "내 포트폴리오 종목들 오늘 어떠니?", "최근 시장 뉴스 알려줘"처럼 물어보면, 실시간 데이터와 뉴스까지 알려준답니다. 💼🤖
