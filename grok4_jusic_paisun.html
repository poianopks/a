1. 파이썬 기초 개념 (왕초보 레벨)

파이썬은 간단하고 읽기 쉬운 프로그래밍 언어입니다. Guido van Rossum이 만들었고, 데이터 분석, 웹 개발, 자동화 등에 사용됩니다. 초보자에게 딱 맞아요! 배우기 쉽고, 영어처럼 자연스러운 문법입니다.

왜 최고인가? 무료, 오픈소스, 방대한 라이브러리 (예: 데이터 처리에 pandas). 2025년에도 가장 인기 있는 언어 중 하나입니다.

2. 파이썬 설치 및 Hello World (왕초보 따라하기)
  1. python.org에서 Python 다운로드 (최신 버전 3.12+ 추천).
  2. 설치 후 명령 프롬프트 열고 python --version 입력해 확인.

    print("Hello, World!")

이게 첫 프로그램! 에러 나면 설치 확인하세요.

3. 파이썬 기본 문법: 변수, 데이터 타입, 제어문 (왕초보 따라하기)

변수와 데이터 타입: 이름 = 값. 예: x = 5 # 정수, name = "홍길동" # 문자열, prices = [100, 200] # 리스트

제어문: if 조건: 코드. 예:

if age >= 18:

루프: for i in range(5): print(i) # 0부터 4까지 출력

함수: def add(a, b): return a + b

연습: 위 코드 복사해 실행해 보세요!

4. 파이썬으로 기업 분석: 주식 데이터 가져오기 (yfinance 사용, 왕초보 따라하기)

yfinance 라이브러리로 Yahoo Finance 데이터 가져옵니다. 기업 분석에 유용: 주가, 재무 정보 등.

  1. 설치: pip install yfinance
  2. 코드 예시:
    3. import yfinance as yf

stock = yf.Ticker("AAPL")  # Apple 회사
info = stock.info
print("회사 이름:", info['longName'])
print("현재 가격:", info['currentPrice'])

history = stock.history(period="1y")  # 1년 데이터
print(history.head())  # 첫 5줄 출력
  3. 실행: python 파일.py. 데이터 분석: 평균 주가 계산 등 (pandas 사용).

분석 팁: history['Close'].mean()으로 평균 주가. 투자 조심!

5. 파이썬으로 뉴스 가져오기: News API 사용 (왕초보 따라하기)

NewsAPI.org로 최신 뉴스 가져옵니다. API 키 필요 (newsapi.org에서 무료 등록).

  1. 설치: pip install requests (또는 urllib 사용)
  2. 코드 예시 (requests 사용):
    3. import requests

api_key = 'your_api_key_here'  # 당신의 키 입력
url = f'https://newsapi.org/v2/top-headlines?country=us&category=business&apiKey={api_key}'
response = requests.get(url)
data = response.json()

for article in data['articles'][:5]:  # 상위 5개
    print(article['title'])
  3. urllib 버전 (설치 없이):
    4. import urllib.request
import json

api_key = 'your_api_key_here'
url = f'https://newsapi.org/v2/top-headlines?country=us&category=business&apiKey={api_key}'
with urllib.request.urlopen(url) as response:
    data = json.loads(response.read().decode())
    for article in data['articles'][:5]:
        print(article['title'])
  4. 실행: 뉴스 제목 출력. 분석: 키워드 검색 등 추가.

더 공부: newsapi-python 패키지 사용.
